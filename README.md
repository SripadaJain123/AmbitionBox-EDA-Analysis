# AmbitionBox EDA Analysis

## Project Overview
This project performs Exploratory Data Analysis (EDA) on the Top 500 companies listed on AmbitionBox to uncover hiring trends, employee satisfaction, workplace ratings, and industry insights across India.

## Objective
- Scrape company data from AmbitionBox
- Clean and preprocess the dataset
- Analyze hiring trends and employee reviews
- Visualize insights using charts and graphs
- Identify top industries and companies

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Beautiful Soup
- Requests
- Jupyter Notebook

## Dataset Features
- Company Name
- Industry Type
- Job Openings
- Work-Life Balance
- Job Security
- Promotions
- Overall Rating
- Employee Reviews
- Office Locations

## Key Insights
- IT Services & Consulting dominates hiring.
- Bangalore, Mumbai, and Hyderabad are major job hubs.
- Accenture has the highest job openings.
- Employee job security ratings are generally high.
- Promotion satisfaction is comparatively lower.

## Project Workflow
1. Web Scraping
2. Data Cleaning
3. Feature Engineering
4. Univariate Analysis
5. Bivariate Analysis
6. Data Visualization
7. Business Insights
